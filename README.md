# Smart Web Finder

Simple project that helps find websites by need. Assets are organized under `assets/`.

How to push to GitHub:

1. Create a remote repository on GitHub (web UI or `gh repo create`).
2. Add remote: `git remote add origin https://github.com/<your-username>/<repo>.git`
3. Push: `git push -u origin main`
