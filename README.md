# Smart Web Finder

A lightweight web app for discovering useful websites based on what you need, such as design, coding, learning, video editing, and productivity.

## Features

- Search websites by need
- Filter by category, pricing, and difficulty
- Clean responsive layout
- Organized project structure with assets separated by type

## Project Structure

- `index.html` — main page
- `assets/css/styles.css` — styles
- `assets/js/script.js` — app logic
- `assets/img/website.jfif` — image asset

## How to Run Locally

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Use the search and filter controls to explore the website list.

## Publish to GitHub

If you want to publish updates after editing:

```bash
git add .
git commit -m "Update README"
git push
```

## GitHub Pages

To make the site available online:

1. Go to the repository on GitHub.
2. Open **Settings**.
3. Select **Pages**.
4. Set the source to the `main` branch and the root folder.
5. Save and wait for the deployment link.

## Notes

- Keep new images in `assets/img/`.
- Keep new styles in `assets/css/`.
- Keep new scripts in `assets/js/`.
