# Smart Web Finder

Smart Web Finder is a static web app that helps users quickly discover legitimate websites based on intent (learning, media, productivity, coding, travel, etc.).

It is designed for fast exploration: select filters, type a need, and get useful links without account walls or heavy setup.

## What it does

- Keyword search across website name, description, and tags
- Multi-filter selection from a wide list of needs
- "Search by filters" button that appears only when filters are selected
- "Clear search" (X) inside the search bar
- Results section that stays hidden until a valid search
- Floating scroll helper:
	- `↓` scrolls to results
	- then changes to `↑` to scroll back to top

## Current filter coverage

Includes categories such as:

- Movies & Series (legal sources)
- Video editing, coding, study
- Jobs, AI tools, finance, health
- Productivity, communication, freelancing
- Content creation, developer tools, design
- Utilities, travel, shopping, entertainment, gaming
- Photo editing, converters, hosting (static/backend)

## Project structure

- `index.html` — app UI and filter controls
- `assets/css/styles.css` — full styling
- `assets/js/script.js` — search/filter logic + dataset
